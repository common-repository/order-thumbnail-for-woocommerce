Adds a thumbnail to orders list in WooCommerce.

== Installation ==

1. Upload `order-thumbnail-for-woocommerce` folder to the `/wp-content/plugins/` directory
2. Activate Order Thumbnail for WooCommerce through the 'Plugins' menu in WordPress
3. Go to the WooCommerce Orders page

It just works!
